Sistema de Cifrado Pro – Análisis Funcional y Lógico del Código Fuente

1. Propósito del Sistema

El Sistema de Cifrado Pro es una aplicación web que implementa algoritmos clásicos de criptografía monoalfabética, específicamente los métodos César y Atbash.

Su objetivo es permitir al usuario cifrar y descifrar mensajes utilizando un alfabeto personalizable y aplicando aritmética modular para garantizar circularidad matemática.

2. Gestión del Universo de Caracteres (PRESETS)

El sistema define alfabetos preconfigurados almacenados en un objeto constante llamado PRESETS. Cada clave representa un conjunto de caracteres. 

const PRESETS = {
    ascii: " !\"#$%&'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNOPQRSTUVWXYZ...",
    az: "ABCDEFGHIJKLMNOPQRSTUVWXYZ",
    minus: "abcdefghijklmnopqrstuvwxyz",
    az_minus: "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz",
    num: "0123456789"
};

El tamaño del conjunto define el valor del Módulo (M), que es esencial en la aritmética modular.

3. Flujo General del Sistema

1. Usuario selecciona o escribe un conjunto.

2. Se calcula automáticamente M = longitud del conjunto.

3. Usuario escribe el mensaje.

4. Selecciona método (César o Atbash).

5. Se ejecuta la función procesar().

6. Se muestra el resultado transformado.

4. Función Principal: procesar(modo)

function procesar(modo) {

    const texto = document.getElementById("texto").value;
    const tipo = document.getElementById("tipo").value;
    let shift = parseInt(document.getElementById("shift").value) || 0;
    const conjunto = inputConjunto.value;

    const M = conjunto.length;
    shift = shift % M;

    let res = "";

    for (let i = 0; i < texto.length; i++) {

        let char = texto[i];
        let idx = conjunto.indexOf(char);

        if (idx !== -1) {

            if (tipo === "cesar") {
                let s = (modo === "descifrar") ? -shift : shift;
                let nuevoIdx = (idx + s) % M;
                if (nuevoIdx < 0) nuevoIdx += M;
                res += conjunto[nuevoIdx];

            } else {
                res += conjunto[(M - 1) - idx];
            }

        } else {
            res += char;
        }
    }

    document.getElementById("resultado").value = res;
}

Descripción Técnica

La función recorre el texto carácter por carácter. Si el carácter pertenece al conjunto, aplica el algoritmo seleccionado. Si no pertenece, lo deja intacto.

5. Lógica Matemática del Sistema

5.1 Cifrado César

E(x) = (x + k) mod M

Donde:
• x = índice del carácter
• k = desplazamiento
• M = tamaño del alfabeto

5.2 Descifrado César

D(x) = (x - k) mod M

5.3 Cifrado Atbash

E(x) = (M - 1) - x

Atbash es simétrico, por lo tanto el mismo proceso sirve para cifrar y descifrar.

7. Manual de Uso

Paso 1: Seleccione el conjunto de caracteres.

Paso 2: Ingrese el mensaje.

Paso 3: Seleccione el método.

Paso 4: Si usa César, defina el salto.

Paso 5: Presione CIFRAR o DESCIFRAR.

Paso 6: Para revertir, use el mismo conjunto y salto.

8. Conclusión Técnica

El sistema implementa correctamente principios de aritmética modular y transformación monoalfabética. Su arquitectura es lineal, eficiente y de complejidad O(n), donde n es la longitud del mensaje.
